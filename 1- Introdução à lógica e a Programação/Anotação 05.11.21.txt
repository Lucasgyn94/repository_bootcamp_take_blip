## Tipos de variaveis
As variáveis e as constantes podem ser classificadas basicaente de quatro tipos: Núméricas: inteiro 
Caracteres: letras alfanúmericas: letras e números 
Lógicas: verdadeiro ou falso

Constantes
As constantes são valores imutáveis que não são alterados até o final do programa

Ex: 
INICIO
// constante
DECLARA PI = 3,14
// variavel
DECLARA raio: numero
FIM

## Expressões Aritméticas
"São expressões que utilizam operadores aritméticos e funções aritméticas envolvendo constantes e variáveis"

Exemplo:
-50 + 50

-total + 50

## Operadores aritméticos
soma: +
subtração: -
multiplicação: *
divisão: /
porcentagem: %

## Expressões literais
"São expressões com constantes e/ou variáveis que tem como resultado valores literais. Iremos utilizar as expressões literais na atribuição de valor para uma variável ou constante"

Exemplos:
nome = "Lucas Ferreira da Silva"
nome <- "Lucas Ferreira da Silva"
media = (nota1+nota2+nota3+nota4)/4

OBS: Sempre que se for exibir valores alfanúmericos, deve se colocar entre aspas.

Exemplos de expressões Aritméticas e Literais

// Variaveis
A, B, C

// Comando de atribuição / operação
A = 2
// Procedimento
Armazenar o valor 2 na variável A

// Comando de atribuição / operação
B = A + 3
// Procedimento
Somar o valor de A(2) com 3 e armazenar em B(5)

// Comando de atribuição / operação
C = A + B
// Procedimento
Somar o valor de A(2) e o valor de B(5) e armazenar em C(7)

## Operadores relacionais
"São expressões compostas por outras expressões ou variáveis númericas com operadores relacionais. As expressões relacionais retornam valores lógicos (verdadeiro/falso)".
Ex: 1 > 2 ? falso
    1 < 2 ? verdadeiro

Os operadores relacionais são representados por símbolos
- Maior que: >
x > y
x e maior que y?

- Maior ou igual: >=
x >= y
x é maior ou igual a y?

- Menor que: <
x < y
x é menor que y?

- Menor ou igual: <=
x <=
x é menor ou igual a y?

- Igualdade: ==
x == y
x é igual a y?

- Diferente de: !=
x != y
x é diferente de y?
